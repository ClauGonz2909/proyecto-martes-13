ESTILOS DE TEXTO

  1. FONT-FAMILY: Es una propiedad que cambia el tipo de tipografia. h1 { font-family:Arial;}
  2. FONT-SIZE: Es un propiedad que cambia el tama√±o de la fuente. h1 { font-size:16px;}
  3. LINE-HEIGHT: Es un propiedad que cambia la distancia entre lineas. h1 { line-height:20px;}
  4. TEXT-ALIGN: Es una propiedad que permite mover el texto( left-center-right) h1 { text-align:center;}
  5. FONT-WEIGHT:Es una propiedad que permite cambiar la intensidad del texto. h1 { font-weight:300;}
  6. TEXT-DECORATION: Es una propiedad que permite subrayar un texto. h1 { text-decoration:none;}

ESTILOS DE TEXTO

  1. SELECTOR DE ETIQUETA: Es el elemento de html que se le va aplicar propiedades y valores.

  2. SELECTOR DE CLASES: Se le puede aplicar estilos usando punto (.) a elementos de html.

    .selector-de-clase{ color:red; }
  3. SELECTOR DE ID: Se le puede aplicar estilos usando punto (#) a elementos de html.

    #selector-id{ background:grey }
  4. SELECTOR DE DESCENDENCIA: Es para aplicar propiedades a elementos que comparten una relacion de descencia (padre e hijo)

    soy un parrafo

    section div p { text-align:center; }

    parrafo prueba 2

     section + p{ color:blue }
