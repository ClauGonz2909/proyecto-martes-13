SELECTORES

1. SELECTOR DE ETIQUETA: Es el elemento de html que se le va aplicar propiedades y valores.
2. SELECTOR DE CLASES: Se le puede aplicar estilos usando punto (.) a elementos de html.
3. SELECTOR DE ID: Se le puede aplicar estilos usando punto (#) a elementos de html.
4. SELECTOR DE DESCENDENCIA: Es para aplicar propiedades a elementos que comparten una   relacion de descencia (padre e hijo)
5. SELECTOR DE HIJO DIRECTO: Se le aplica propiedades a todos los hijos menos lo que no son directos.
6. SELECTOR HERMANO SIGUIENTE O ADYANCENTE: Se le aplica propiedades al elemento que sigue despues 
7. SELECTOR DE HERMANOS SIGUIENTES O ADYACENTES: Se le aplica propiedad a todos los elementos que siguen.
8. SELECTOR DE ANIDAMIENTO: Es para especificar donde quieres aplicar la propiedad dentro de un elemento.
